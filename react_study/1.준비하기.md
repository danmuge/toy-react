### javaScript 문법

**자료형**

- **변수의 데이터가 대입되는 시점에 자료형이 결정 (동적 타이핑)**
- **Number, String, Boolean, Null(값이 널), Undefined(값이 지정X), Array**
- **Object(객체)**

**연산자**

- **대입 연산자**
    - **let a = 10;**
- **산술 연산자**
    - **+ - * /  %  **(지수 연산자)**
    - **+=  -=   *=   /=**
- **증가,감소 연산자 (++, --)**
    - **a++,++a,--b,b--**
- **비교 연산자**
    - **<,>,<=,>=**
    - **==,!=**
    - **===, !== (자료형 까지 비교)**
    - **&&(And),||(OR)**
    - **a ? 1 : 2 (삼항 연산자)**
